# Fonts folder

Place `Inter-Variable.woff2` here to enable self-hosted Inter for both screen and print.

Recommended source: https://github.com/rsms/inter/releases

- File name: `Inter-Variable.woff2`
- Path after adding: `/public/fonts/Inter-Variable.woff2`
- Already wired via @font-face in `src/index.css`

After copying, rebuild/dev server will serve it at `/fonts/Inter-Variable.woff2`.

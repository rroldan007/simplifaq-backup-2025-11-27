# App Brand Assets

Coloca tus logos en esta carpeta. Archivos requeridos:

## Logos de la Barra Lateral
- **logo-light.png** - Logo para modo claro (fondo claro)
- **logo-dark.png** - Logo para modo oscuro (fondo oscuro)
  - Recomendación: altura 32-40px, ancho automático
  - Formato: PNG con fondo transparente
  - El logo se mostrará en la barra lateral izquierda

## Iconos de la Aplicación (Opcionales)
- app-icon.png (icono cuadrado principal, 512x512 recomendado)
- icon-192.png (icono PWA)
- icon-512.png (icono PWA)
- logo.svg (logo vectorial opcional)

## Referencias usadas por la app:
- **Sidebar Logo (Modo Claro):** /brand/logo-light.png
- **Sidebar Logo (Modo Oscuro):** /brand/logo-dark.png
- Favicon: /brand/app-icon.png (index.html)
- Apple Touch Icon: /brand/app-icon.png (index.html)
- Web Manifest icons: /brand/icon-192.png y /brand/icon-512.png (site.webmanifest)

## Notas:
- Puedes reemplazar estos archivos en cualquier momento sin cambiar código.
- Si no existen los logos, se mostrará automáticamente un logo de texto "S Simplifaq" como fallback.
- Para mejor resultado, usa PNG con fondo transparente.
- El logo-dark.png debe tener colores claros para verse bien en fondo oscuro.
- El logo-light.png debe tener colores oscuros para verse bien en fondo claro.

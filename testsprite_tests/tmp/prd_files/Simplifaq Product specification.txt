Product Specification & Functional Description
1. Product Purpose

Simplifaq is a SaaS invoicing application built exclusively for the Swiss market.
Its purpose is to allow independent professionals and small businesses to create, manage, send, and track invoices in full compliance with Swiss fiscal regulations, while remaining simple, fast, and accessible to non-accountants.

The product focuses on:

Legal compliance (TVA, Swiss QR Bills)

Ease of use (clean UI, minimal steps)

SaaS-level visibility (metrics, dashboards, usage tracking)

Scalability (subscriptions, Stripe integration)

Simplifaq is not a full accounting suite. It is an invoicing-first SaaS, designed to evolve.

2. Target Users
Primary Users

Swiss independents (entreprise individuelle)

Freelancers

Small service businesses

Micro-entrepreneurs issuing repetitive invoices

Internal Users

Admin (Simplifaq team)
Needs full visibility on users, subscriptions, invoices, and SaaS KPIs.

7. How the System Should Work (Flow Summary)

User signs up

User subscribes via Stripe

Subscription activates account

User creates clients and invoices

System generates compliant Swiss invoice + QR Bill

Invoice sent as PDF

TVA automatically tracked

Admin monitors usage, users, subscriptions

Admin can invoice users directly if needed
